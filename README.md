# React Redux 
This is a WebAPP clone built using [React.js](https://reactjs.org), [Redux](https://redux.js.org), and [Node.js](https://nodejs.org).

# Technologies

1. [React.js](https://reactjs.org)
1. [Redux](https://redux.js.org)
1. [create-react-app](https://github.com/facebook/create-react-app)
1. [Express.js](https://expressjs.com)
1. [Node.js](https://nodejs.org)
1. [MongoDB](https://mongodb.com)

### Features
- Write a story
- Social Sign-in
- View a story
- View all stories
- Follow a user
- Clap a story

### Installation
* Git clone this repo [here](https://github.com/GhostGeneral/React-redux-Nodejs).
* Move into the folder from your terminal:
```sh
    cd React-Redux-Nodejs
```
* Run `npm install` to install node dependencies.
* Start MongoDB instance: `mongod`.
* Run `npm run dev` to start the `nodeidon` daemon.
